# Paradise-of-Drin

## Description
**Paradise of Drin** is a static, visually engaging tourism website designed to showcase the natural beauty and attractions of the Drin region. The project focuses on a clean, modern design with interactive elements to enhance user experience and provide an appealing digital presentation of the region’s landscapes and activities.

## Features
- **Full-page background image:** A high-quality scenic image covers the entire viewport, creating an immersive visual experience.  
- **Responsive navigation bar:** Green top navigation bar with a red highlighted "Home" button and additional menu items featuring hover effects and smooth transitions.  
- **Interactive side menu:** Vertical gray menu with clickable icons, which move slightly on hover to indicate interactivity.  
- **Custom typography and styling:** Large, italicized headings with Times New Roman and stylized fonts to highlight the site’s title.  
- **Hover and shadow effects:** All menu elements have hover effects and subtle shadows to create depth and responsiveness.  
- **Flexible layout:** Uses CSS Flexbox for clean alignment and spacing.

## Technologies Used
- HTML5 – for structure and semantic content  
- CSS3 – for styling, hover effects, and responsive design  
- Images – background and icons to enrich the interface visually
